## 4. FAQ

### Perché il widget non si aggiorna subito?

Il sistema decide il ritmo di aggiornamento dei widget, quindi il refresh potrebbe non essere immediato. Apri l’app per verificare che l’elemento sia stato salvato, torna alla schermata Home e attendi l’aggiornamento di sistema oppure aggiungi di nuovo il widget.

### I miei dati vengono caricati su un server?

No. La versione attuale salva i dati localmente per impostazione predefinita, non richiede un account e non offre sincronizzazione cloud.

### Posso fare un backup dei miei dati?

Sì. Puoi esportare un file JSON dalle impostazioni e importare o unire i dati più tardi da un file JSON.

### Come elimino tutti i dati?

Usa “Cancella tutti i dati” nelle impostazioni oppure elimina l’app. Se hai esportato dei backup, elimina anche quei file separatamente.
