# Informativa sulla privacy di AwaitGoods (Italiano)

Data di validità: 2026-05-02

Usando AwaitGoods, accetti il trattamento dei dati descritto in questa informativa sulla privacy.

AwaitGoods è una wishlist e lista acquisti orientata al salvataggio locale. Trattiamo solo le informazioni necessarie alle funzioni principali dell’app.

## 1. Chi siamo

- Nome del prodotto: AwaitGoods
- Sito sviluppatore: https://mrqk.com
- Email: yokinzhu@gmail.com
- WeChat: zhuyokin

## 2. Quali informazioni trattiamo

Le seguenti informazioni possono essere salvate localmente sul dispositivo:

- Elementi creati dall’utente, come nome, prezzo, importo risparmiato, link, categoria, note, stato, priorità e colore di contrassegno
- Ordinamento, data di creazione e aggiornamento
- Preferenze come modalità aspetto, palette del tema, lingua e widget
- File JSON che scegli di esportare o importare

## 3. Come usiamo queste informazioni

Queste informazioni servono solo a:

- mostrare e gestire la wishlist e la lista acquisti
- calcolare e mostrare il progresso del risparmio
- mostrare riepiloghi degli elementi in attesa nei widget
- supportare ricerca, filtri, ordinamento, esportazione e importazione
- salvare preferenze di lingua e aspetto

## 4. Cosa non facciamo

Nella versione attuale, AwaitGoods:

- non richiede account
- non carica i dati della wishlist su server dello sviluppatore
- non vende dati personali
- non usa SDK pubblicitari di tracciamento
- non condivide dati senza un’azione volontaria dell’utente

## 5. Autorizzazioni

Le funzioni principali della versione attuale non richiedono permessi aggiuntivi per pubblicità, marketing o promozione. Se versioni future aggiungeranno promemoria opzionali, anche queste notifiche saranno attivate solo localmente sul dispositivo.

## 6. Widget

Se usi i widget della schermata Home, l’app scrive un riepilogo limitato dei tuoi elementi nel contenitore App Group consentito dal sistema. Questi dati restano sul tuo dispositivo e non vengono caricati su server esterni.

## 7. Esportazione, importazione e condivisione

Quando esporti, l’app crea un file JSON sul tuo dispositivo. Decidi tu dove salvarlo, inviarlo o conservarlo. L’importazione legge solo il file JSON che scegli esplicitamente.

## 8. Link esterni

Puoi salvare link a prodotti in AwaitGoods. Aprire un link può avviare un sito web o un’app di terze parti, che seguono proprie pratiche di privacy.

## 9. Informazioni quando ci contatti

Se contatti il supporto via email o WeChat, riceviamo solo le informazioni che scegli di fornire, come contatti, descrizione del problema, screenshot o dati sul dispositivo. Le usiamo solo per rispondere, gestire feedback e migliorare il prodotto.

## 10. Conservazione ed eliminazione dei dati

I dati di AwaitGoods vengono salvati localmente sul tuo dispositivo per impostazione predefinita.

Puoi eliminare i dati:

- cancellando tutti i dati nelle impostazioni dell’app
- eliminando singoli elementi
- disinstallando l’app dal dispositivo
- cancellando manualmente i file di backup esportati

## 11. Privacy dei minori

AwaitGoods non è rivolto ai minori e non raccoglie consapevolmente dati personali di minori.

## 12. Aggiornamenti della policy

Se versioni future introdurranno account, sincronizzazione cloud, analisi o SDK di terze parti, aggiorneremo questa informativa e spiegheremo i cambiamenti in modo ragionevole.
