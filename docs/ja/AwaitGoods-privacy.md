# AwaitGoods プライバシーポリシー（日本語）

施行日：2026-05-02

AwaitGoods を利用することで、本ポリシーに記載されたデータの取り扱いに同意したものとみなされます。

AwaitGoods は、本地保存を優先する wishlist / 買い物リスト App です。データ処理は最小限にし、App の基本機能に必要な情報だけを扱います。

## 2. 処理する情報

以下の情報が端末内に保存される場合があります：

- 作成した候物項目。名前、価格、貯金額、リンク、カテゴリ、メモ、状態、優先度、マーク色など
- 並び順、作成日時、更新日時
- 外観モード、テーマ配色、言語、ウィジェット表示などの設定
- ユーザーが書き出し、または読み込む JSON バックアップファイルの内容

## 3. 情報の利用目的

これらの情報は次の目的にのみ使用します：

- wishlist と買い物リストの表示と管理
- 貯金進捗の計算と表示
- ウィジェットで候物の概要を表示
- 検索、絞り込み、並び替え、書き出し、読み込みの提供
- 言語と外観設定の保存

## 4. 行わないこと

現在のバージョンで AwaitGoods は以下を行いません：

- アカウント登録の要求
- 候物データの開発者サーバーへのアップロード
- 個人情報の販売
- 広告トラッキング SDK によるプロファイリングや App 間追跡
- ユーザーの操作なしでのデータ共有

## 5. 権限について

現在の主要機能では、広告、マーケティング、プロモーションのための追加権限は必要ありません。将来のバージョンで任意のリマインダー機能が追加される場合でも、その通知は端末内でのみ発火します。

## 6. ウィジェット

ホーム画面ウィジェットを使う場合、App は一部の候物概要をシステムが許可する App Group 共有コンテナに書き込みます。データは端末内に残り、外部サーバーへアップロードされません。

## 7. 書き出し、読み込み、共有

書き出し機能を使うと、App は端末上に JSON ファイルを作成します。そのファイルをどこに保存、送信、バックアップするかはユーザーが決めます。読み込み機能は、ユーザーが選択した JSON ファイルだけを読み取ります。

## 8. 外部リンク

AwaitGoods には商品リンクを保存できます。リンクを開くと、第三者のウェブサイトや App が開く場合があります。これらのサービスは第三者が提供し、それぞれのプライバシーポリシーに従います。

## 9. お問い合わせ時の情報

メールまたは WeChat でサポートに連絡した場合、連絡先、問題内容、スクリーンショット、端末情報など、ユーザーが提供した情報を受け取ります。これらは返信、フィードバック対応、製品改善のためだけに使用します。

## 10. データの保存と削除

AwaitGoods のデータは既定で端末内に保存されます。

データは以下の方法で削除できます：

- App 設定で全データを削除
- 個別の候物項目を削除
- 端末から App をアンインストール
- 書き出したバックアップファイルを自分で削除

## 11. 子どものプライバシー

AwaitGoods は子どもを対象として設計されておらず、子どもの個人情報を意図的に収集しません。

## 12. ポリシーの更新

将来、アカウント、クラウド同期、分析、第三者 SDK を追加する場合、本ポリシーを更新し、合理的な方法で変更を説明します。
