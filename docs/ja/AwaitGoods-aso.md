## App 説明

AwaitGoods は、本地保存を軸にした願いごとリスト / 買い物リスト App です。欲しいものを先に記録し、価格、貯金額、状態、優先度を見ながら、本当に持ちたいかをゆっくり判断できます。

主な体験：

- 名前、価格、貯金額、カテゴリ、リンク、メモ、優先度、状態を記録
- ホームで検索、絞り込み、手動並び替え、一括整理、貯金進捗を確認
- 詳細画面で貯金額を追加、または一括で満額にする
- ホーム画面ウィジェットで wishlist の概要と進捗を表示
- JSON の読み込みと書き出しでオフラインバックアップ
- 和風ミニマル、クリーム系ミニマル、Apple ライクなミニマルなど複数テーマを切り替え
- 簡体字中国語、繁体字中国語、英語、日本語、韓国語、ドイツ語、イタリア語、フランス語に対応
- ライト、ダーク、システム連動の外観モードに対応
- オンボーディング、Launch Screen、サポートページ、プライバシーポリシーページを提供

AwaitGoods は EC サービスでも複雑な家計簿でもありません。静かな wishlist として、気持ちを一度置いてから、納得できる買い方を選べるようにします。
