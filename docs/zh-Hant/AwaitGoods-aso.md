## 應用描述

候物 AwaitGoods 是一款本地優先的心願清單與購物清單工具，幫助使用者把「想買」先放進列表，再透過價格、已存金額和狀態判斷是否值得繼續靠近。

核心體驗包括：

- 記錄名稱、價格、已存金額、分類、連結、備註、優先級與狀態
- 從首頁查看搜尋、篩選、手動排序、批量整理和存錢進度
- 在詳情頁繼續存入金額或一鍵補滿目標金額
- 使用桌面小工具查看心願摘要與進度
- 使用 JSON 匯入與匯出做離線備份
- 切換多套主題風格，支援日系清新、奶油極簡、Apple 極簡等視覺方向
- 支援簡體中文、繁體中文、English、日本語、한국어、Deutsch、Italiano、Français
- 支援淺色、深色與跟隨系統外觀
- 提供首次安裝引導頁、Launch 啟動頁、技術支援頁與隱私權政策頁

候物不是電商，也不是複雜帳本。它更像一個安靜的 wishlist：先記下想買，再慢慢判斷，最後把真正值得擁有的東西帶回生活裡。
