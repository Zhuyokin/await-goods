# 候物 AwaitGoods 隱私權政策（繁體中文）

生效日期：2026-05-02

如果你使用候物 AwaitGoods，代表你同意本隱私權政策中所描述的資料處理方式。

候物 AwaitGoods 是一款以本地記錄為主的心願清單與購物清單應用。我們會盡量少處理資料，只在實現產品核心功能所必需的範圍內使用資訊。

## 2. 我們會處理哪些資訊

在你使用 App 過程中，以下內容可能保存在你的裝置上：

- 你建立的候物條目，例如名稱、價格、已存金額、連結、分類、備註、狀態、優先級與標記色
- 排序資訊、建立時間與更新時間
- 外觀模式、主題配色、語言與小工具展示等偏好設定
- 你主動匯出或匯入的 JSON 備份檔案內容

## 3. 我們如何使用這些資訊

上述資訊僅用於以下目的：

- 展示和管理你的心願清單與購物清單
- 計算並顯示存錢進度
- 在小工具中顯示等待中的候物摘要
- 支援搜尋、篩選、排序、匯出與匯入
- 保存語言與介面偏好設定

## 4. 我們不會做的事情

目前版本中，候物 AwaitGoods 不會：

- 要求你註冊帳號
- 將你的候物資料上傳到開發者伺服器
- 出售你的個人資訊
- 使用廣告追蹤 SDK 對你進行描繪或跨 App 追蹤
- 在未經你主動操作的情況下分享你的資料

## 5. 權限說明

目前版本的核心功能不需要為了廣告、行銷或推廣請求額外權限。若未來版本加入可選提醒功能，相關通知也只會在你的裝置本地觸發。

## 6. 小工具說明

如果你使用桌面小工具，App 會把部分候物摘要寫入系統允許的 App Group 共享容器，方便小工具讀取。這些資料仍保存在你的裝置本地，不會上傳到外部伺服器。

## 7. 匯出、匯入與分享

當你主動使用匯出功能時，App 會在裝置上生成 JSON 檔案。檔案如何保存、傳送或備份由你自行決定。匯入功能只會讀取你主動選擇的 JSON 檔案。

## 8. 外部連結

你可以在候物中保存商品連結。點擊連結後，系統可能開啟第三方網站或第三方 App。這些服務由第三方提供，並有各自的隱私權政策與資料處理方式。

## 9. 聯絡我們時的資訊

如果你透過信箱或微信聯絡技術支援，我們會收到你主動提供的資訊，例如聯絡方式、問題描述、截圖或裝置資訊。這些資訊只用於回覆諮詢、處理回饋與改進產品體驗。

## 10. 資料保存與刪除

你的候物資料預設保存在裝置本地。

你可以透過以下方式刪除資料：

- 在 App 設定中清空全部資料
- 刪除單個候物條目
- 從裝置中卸載 App
- 自行刪除已匯出的備份檔案

## 11. 未成年人隱私

候物 AwaitGoods 不面向兒童定向設計，也不會故意收集兒童個人資訊。

## 12. 隱私權政策更新

如果未來接入帳號、雲端同步、統計分析或第三方 SDK，我們會同步更新本隱私權政策，並透過合理方式說明變化。
