## 4. FAQ

### Pourquoi le widget ne se rafraîchit-il pas immédiatement ?

Le système gère le rythme de mise à jour des widgets, ce qui peut retarder l’affichage. Ouvrez l’app pour vérifier que l’élément est bien enregistré, revenez à l’écran d’accueil et attendez le rafraîchissement système ou ajoutez à nouveau le widget.

### Mes données sont-elles envoyées vers un serveur ?

Non. La version actuelle stocke les données localement par défaut, ne demande pas de compte et n’offre pas de synchronisation cloud.

### Puis-je sauvegarder mes données ?

Oui. Vous pouvez exporter un fichier JSON depuis les réglages et importer ou fusionner plus tard des données à partir d’un fichier JSON.

### Comment supprimer toutes les données ?

Utilisez « Effacer toutes les données » dans les réglages ou supprimez l’app. Si vous avez exporté des sauvegardes, supprimez aussi ces fichiers.
