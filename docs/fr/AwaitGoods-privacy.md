# Politique de confidentialité AwaitGoods (Français)

Date d’effet : 2026-05-02

En utilisant AwaitGoods, vous acceptez le traitement des données décrit dans cette politique de confidentialité.

AwaitGoods est une app locale de wishlist et de liste d’achats. Nous limitons le traitement des informations à ce qui est nécessaire aux fonctionnalités principales.

## 1. Qui nous sommes

- Nom du produit : AwaitGoods
- Site développeur : https://mrqk.com
- E-mail : yokinzhu@gmail.com
- WeChat : zhuyokin

## 2. Quelles informations nous traitons

Les informations suivantes peuvent être enregistrées localement sur votre appareil :

- Les éléments que vous créez, comme le nom, le prix, le montant épargné, le lien, la catégorie, les notes, le statut, la priorité et la couleur de marque
- L’ordre de tri, la date de création et la date de mise à jour
- Les préférences comme le mode d’apparence, la palette de thème, la langue et l’affichage des widgets
- Les fichiers JSON que vous choisissez d’exporter ou d’importer

## 3. Comment nous utilisons ces informations

Ces informations servent uniquement à :

- afficher et gérer votre wishlist et votre liste d’achats
- calculer et afficher la progression d’épargne
- montrer un résumé des éléments en attente dans les widgets
- permettre la recherche, les filtres, le tri, l’export et l’import
- enregistrer vos préférences de langue et d’apparence

## 4. Ce que nous ne faisons pas

Dans sa version actuelle, AwaitGoods :

- ne demande pas de compte
- n’envoie pas vos données de wishlist vers un serveur développeur
- ne vend pas vos informations personnelles
- n’utilise pas de SDK publicitaire de suivi
- ne partage pas vos données sans action volontaire de votre part

## 5. Autorisations

Les fonctions principales de la version actuelle ne demandent pas d’autorisations supplémentaires pour la publicité, le marketing ou la promotion. Si de futures versions ajoutent des rappels optionnels, ces notifications seront elles aussi déclenchées localement sur votre appareil.

## 6. Widgets

Si vous utilisez des widgets d’écran d’accueil, l’app écrit un résumé limité de vos éléments dans le conteneur App Group autorisé par le système. Ces données restent sur votre appareil et ne sont pas envoyées vers des serveurs externes.

## 7. Export, import et partage

Lorsque vous exportez, l’app crée un fichier JSON sur votre appareil. Vous choisissez vous-même où le sauvegarder, l’envoyer ou le conserver. L’import lit uniquement le fichier JSON que vous sélectionnez explicitement.

## 8. Liens externes

Vous pouvez enregistrer des liens produits dans AwaitGoods. L’ouverture d’un lien peut lancer un site web ou une app tierce, qui dispose de ses propres pratiques de confidentialité.

## 9. Informations lorsque vous nous contactez

Si vous contactez le support par e-mail ou WeChat, nous recevons uniquement les informations que vous choisissez de fournir, comme vos coordonnées, la description du problème, des captures d’écran ou des informations sur l’appareil. Nous les utilisons pour répondre, traiter les retours et améliorer le produit.

## 10. Conservation et suppression des données

Vos données AwaitGoods sont stockées localement sur votre appareil par défaut.

Vous pouvez supprimer les données en :

- effaçant toutes les données dans les réglages de l’app
- supprimant des éléments individuellement
- désinstallant l’app de votre appareil
- supprimant vous-même les fichiers de sauvegarde exportés

## 11. Confidentialité des enfants

AwaitGoods n’est pas destiné aux enfants et ne collecte pas sciemment d’informations personnelles concernant des enfants.

## 12. Mise à jour de cette politique

Si de futures versions ajoutent des comptes, une synchronisation cloud, des analyses ou des SDK tiers, nous mettrons à jour cette politique et expliquerons les changements de manière raisonnable.
