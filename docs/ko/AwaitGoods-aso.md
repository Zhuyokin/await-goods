## 앱 설명

AwaitGoods는 사고 싶은 마음을 로컬 중심의 조용한 목록으로 정리하는 앱입니다. 먼저 저장하고 가격과 저축액을 기록한 뒤, 진행률과 상태를 보며 정말 가질 만한지 천천히 판단할 수 있습니다.

핵심 경험:

- 이름, 가격, 저축액, 분류, 링크, 메모, 우선순위, 상태 기록
- 홈에서 검색, 필터, 수동 정렬, 일괄 정리, 저축 진행률 확인
- 상세 화면에서 저축액 추가 또는 목표 금액 채우기
- 홈 화면 위젯으로 위시리스트 요약과 진행률 표시
- JSON 가져오기와 내보내기로 오프라인 백업
- 일본풍 미니멀, 크리미 미니멀, Apple 스타일 미니멀 등 여러 테마 전환
- 간체 중국어, 번체 중국어, 영어, 일본어, 한국어, 독일어, 이탈리아어, 프랑스어 지원
- 라이트 모드, 다크 모드, 시스템 모드 지원
- 온보딩, Launch Screen, 기술 지원 페이지, 개인정보 처리방침 페이지 제공

AwaitGoods는 쇼핑몰도 아니고 복잡한 가계부도 아닙니다. 사고 싶은 마음을 잠시 보관해 두고, 정말 필요한지 천천히 확인할 수 있게 돕는 조용한 wishlist입니다.

## 이번 버전의 새로운 점

- 첫 실행 온보딩을 추가해 기록, 진행률, 위젯, 백업의 기본 흐름을 소개
- AwaitGoods 그래픽과 앱 이름을 보여 주는 아트형 Launch Screen 추가
- 온보딩 언어 버튼에 자동 순환 힌트를 넣어 언어 전환을 더 쉽게 발견할 수 있도록 개선
- 추가 버튼을 목록 오른쪽 아래의 둥근 플로팅 버튼으로 이동
- 검색을 오른쪽 위 아이콘 뒤로 이동해 홈 상단을 더 깔끔하게 정리
- 긴 제목에서도 우선순위와 가격이 안정적으로 보이도록 목록 행 레이아웃 개선
- 추가, 삭제, 상태 변경 후 위젯 데이터 새로고침 안정성 개선
- 다크 모드를 유지하면서 여러 가지 세련된 테마 팔레트 추가
- 기본 통화 표시를 USD로 변경해 더 국제적인 기본값 제공
- 8개 언어용 App Store 문안, 기술 지원 페이지, 개인정보 처리방침 정리

## 심사 메모

AwaitGoods는 로컬 중심 위시리스트 / 쇼핑리스트 앱입니다. 계정, 인앱 구매, 타사 로그인은 없으며 심사에서는 플로팅 추가 버튼, 오른쪽 위 검색, 필터와 정렬, 상세 화면, 저축 진행률, JSON 가져오기와 내보내기, 위젯, 언어 전환, 테마 전환을 확인할 수 있습니다.
