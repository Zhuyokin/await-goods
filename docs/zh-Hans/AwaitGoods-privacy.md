# 候物 AwaitGoods 隐私政策

生效日期：2026-05-02

如果你使用候物 AwaitGoods，说明你同意本隐私政策中描述的处理方式。

候物 AwaitGoods 是一款以本地记录为主的心愿清单与购物清单应用。我们尽量少收集信息，只在实现产品基本功能所必需的范围内处理数据。

## 2. 我们会处理哪些信息

在你使用 App 过程中，以下内容可能会被保存在你的设备上：

- 你创建的候物条目，如名称、价格、已存金额、链接、分类、备注、状态、优先级、标记色和排序信息
- 你的本地偏好设置，如外观模式、主题配色、语言和小组件展示
- 你主动导出或导入的 JSON 备份文件内容

这些信息默认仅保存在你的设备本地，用于实现 App 的基础功能。

## 3. 我们如何使用这些信息

上述信息仅用于以下目的：

- 展示和管理你的候物清单
- 计算和展示存钱进度
- 在小组件中展示等待中的物品
- 支持搜索、筛选、排序、导出和导入
- 保存你的界面与功能偏好设置

## 4. 我们不会做的事情

当前版本中，候物 AwaitGoods 不会：

- 不会要求你注册账号
- 不会将你的候物数据上传到开发者服务器
- 不会出售你的个人信息
- 不会使用广告追踪 SDK 对你进行画像或跨应用跟踪
- 不会在未经你主动操作的情况下把你的数据分享给第三方

## 5. 权限说明

当前版本的核心功能不需要注册账号，也不会为了广告、营销或推广请求额外权限。若未来版本提供可选提醒能力，相关通知也只会由系统在你的设备本地触发。

## 6. 小组件说明

如果你使用桌面小组件，App 会将部分候物摘要信息写入系统允许的小组件共享容器，以便在小组件中显示等待中的条目。

这些数据仍然保存在你的设备本地，不会因此上传到外部服务器。

## 7. 导出、导入与分享

如果你主动使用导出功能，App 会在你的设备上生成 JSON 文件，供你自行保存或分享。导入功能只会读取你主动选择的 JSON 文件。

导出行为由你主动触发。导出后的文件如何保存、发送或备份，由你自行决定。

## 8. 外部链接

候物 AwaitGoods 允许你保存商品链接。点击链接后，系统可能会打开第三方网站或第三方应用。

这些第三方服务并不由我们控制，它们有自己的隐私政策和数据处理方式。建议你在使用前自行查阅相关说明。

## 9. 联系我们时的信息

如果你通过邮箱或微信联系技术支持，我们会收到你主动提供的信息，例如：

- 邮箱地址或微信身份信息
- 你在反馈中填写的问题描述、截图或设备信息

这些信息仅用于回复你的咨询、处理反馈和改进产品体验，不会用于与你的使用数据做广告画像关联。

## 10. 数据保存与删除

你的候物数据默认保存在你的设备本地。

你可以通过以下方式删除数据：

- 在 App 设置中清空全部数据
- 删除单个候物条目
- 从设备中卸载 App

如果你曾主动导出或分享过文件，你需要自行删除已导出的副本。

## 11. 未成年人隐私

候物 AwaitGoods 不面向儿童定向设计，也不会故意收集儿童个人信息。

如果你认为我们可能在不知情的情况下处理了未成年人信息，请通过下方联系方式联系我们。

## 12. 隐私政策更新

如果未来产品功能发生变化，例如接入账号系统、云同步、统计分析或第三方 SDK，我们会同步更新本隐私政策。

更新后的版本会通过应用内版本更新、网站页面或其他合理方式向你说明。
