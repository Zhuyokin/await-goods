## 应用描述

候物 AwaitGoods 是一款本地优先的心愿清单与购物清单工具，帮助用户把“想买”先放进列表，再通过价格、已存金额和状态判断是否值得继续靠近。

核心体验包括：

- 记录名称、价格、已存金额、分类、链接、备注、优先级与状态
- 从首页查看搜索、筛选、手动排序、批量整理和存钱进度
- 在详情页继续存入金额或一键补满目标金额
- 使用桌面小组件查看心愿摘要与进度
- 使用 JSON 导入与导出做离线备份
- 切换多套主题风格，支持日系清新、奶油极简、Apple 极简等视觉方向
- 支持简体中文、繁体中文、English、日本語、한국어、Deutsch、Italiano、Français
- 支持浅色、深色和跟随系统外观
- 提供首次安装引导页、Launch 启动页、技术支持页和隐私政策页

候物不是电商，也不是复杂账本。它更像一个安静的 wishlist：先记下想买，再慢慢判断，最后把真正值得拥有的东西带回生活里。
