## 홍보 문구

AwaitGoods는 바로 사기보다 먼저 적어 두는 미니멀 위시리스트입니다. 가격, 모은 금액, 분류, 링크와 메모를 남기고 진행률을 보며 더 차분하게 결정하세요.

## 앱 설명

AwaitGoods는 사고 싶은 마음을 가볍고 로컬 중심의 목록으로 정리합니다. 마음에 드는 것을 먼저 저장하고 가격과 저축액을 적은 뒤, 정말 가질 만한지 천천히 판단할 수 있습니다.

주요 기능:

- 가격, 저축액, 분류, 링크와 함께 빠르게 추가
- 홈 목록에서 상태, 우선순위, 저축 진행률 확인
- 상세 화면에서 저축액 추가 또는 목표 채우기
- 오른쪽 위 아이콘에서 여는 검색, 필터, 수동 정렬, 일괄 정리
- 위시리스트 요약과 저축 진행률을 보여주는 홈 화면 위젯
- JSON 내보내기와 가져오기
- 간체 중국어, 번체 중국어, 영어, 일본어, 프랑스어, 독일어, 이탈리아어, 한국어
- 라이트, 다크, 시스템 화면 모드
- 여러 가지 세련된 테마 팔레트

## v1.0.4 새로운 점

- 위시 기록, 저축 진행률, 정리와 백업을 소개하는 더 감성적인 첫 실행 안내 화면
- AwaitGoods 테마 그래픽과 앱 이름을 보여 주는 새로운 아트형 Launch Screen
- 언어 버튼이 자동으로 순환하며 언어 전환 가능성을 알려 주도록 개선
- 추가 버튼을 목록 오른쪽 아래의 둥근 플로팅 버튼으로 이동
- 검색을 오른쪽 위 아이콘 뒤로 이동해 홈 상단을 더 깔끔하게 정리
- 긴 제목에서도 우선순위와 가격이 안정적으로 보이도록 목록 행 레이아웃 개선
- 추가, 삭제, 상태 변경 후 위젯 데이터 동기화 안정성 개선
- 위젯 이름과 표시 문구에 간체 중국어, 번체 중국어, 영어 로컬라이징 추가
- 설정에 여러 가지 세련된 테마 팔레트 추가
- 기본 통화 표시를 USD로 바꿔 더 국제적인 기본값으로 조정
- 설정의 버전 표시를 v1.0.4로 업데이트

## 심사 메모

AwaitGoods는 로컬 중심 위시리스트 / 쇼핑리스트 앱입니다. 계정, 인앱 구매, 타사 로그인은 없으며 심사에서는 플로팅 추가 버튼, 오른쪽 위 검색 아이콘, 필터와 정렬, 상세 화면, 저축 진행률, 위젯, 언어 전환, 테마 전환을 확인할 수 있습니다. 데이터는 기본적으로 기기에 저장됩니다.
