## 4. FAQ

### 위젯이 바로 새로고침되지 않는 이유는 무엇인가요?

위젯 새로고침 시점은 시스템이 조정하므로 즉시 반영되지 않을 수 있습니다. 앱을 열어 항목이 저장됐는지 확인한 뒤 홈 화면으로 돌아가 조금 기다리거나 위젯을 다시 추가해 보세요.

### 데이터가 서버로 업로드되나요?

아니요. 현재 버전은 데이터를 기본적으로 기기에 저장하며 계정을 요구하지 않고 클라우드 동기화도 제공하지 않습니다.

### 데이터를 백업할 수 있나요?

네. 설정에서 JSON 파일을 내보내고 나중에 JSON 파일에서 데이터를 가져오거나 병합할 수 있습니다.

### 모든 데이터를 삭제하려면 어떻게 하나요?

설정의 “모든 데이터 지우기”를 사용하거나 앱을 삭제하세요. 내보낸 백업 파일이 있다면 해당 파일도 따로 삭제해야 합니다.
