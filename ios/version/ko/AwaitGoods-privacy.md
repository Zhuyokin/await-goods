# AwaitGoods 개인정보 처리방침 (한국어)

시행일: 2026-05-02

AwaitGoods를 사용하면 이 개인정보 처리방침에 설명된 데이터 처리 방식에 동의한 것으로 간주됩니다.

AwaitGoods는 로컬 우선 위시리스트 및 쇼핑리스트 앱입니다. 기본 기능에 필요한 최소한의 정보만 처리합니다.

- 제품명: AwaitGoods
- 개발자 웹사이트: https://mrqk.com
- 이메일: yokinzhu@gmail.com

## 2. 어떤 정보를 처리하나요

다음 정보가 기기에 로컬로 저장될 수 있습니다:

- 사용자가 만든 항목: 이름, 가격, 모은 금액, 링크, 분류, 메모, 상태, 우선순위, 표시 색상
- 정렬 정보, 생성일, 수정일
- 화면 모드, 테마 색상, 언어, 위젯 표시 등 선호 설정
- 사용자가 내보내거나 가져오는 JSON 파일

## 3. 이 정보를 어떻게 사용하나요

이 정보는 다음 목적에만 사용됩니다:

- 위시리스트와 쇼핑리스트 표시 및 관리
- 저축 진행률 계산 및 표시
- 위젯에서 대기 중인 항목 요약 표시
- 검색, 필터, 정렬, 가져오기, 내보내기 지원
- 언어와 화면 설정 저장

## 4. 하지 않는 일

현재 버전의 AwaitGoods는 다음을 하지 않습니다:

- 계정 가입 요구
- 위시리스트 데이터를 개발자 서버에 업로드
- 개인정보 판매
- 광고 추적 SDK 사용
- 사용자가 선택하지 않은 데이터 공유

## 5. 권한 안내

현재 버전의 핵심 기능은 광고, 마케팅, 홍보를 위한 추가 권한을 요구하지 않습니다. 추후 선택형 알림 기능이 추가되더라도 해당 알림은 기기 안에서만 작동합니다.

## 6. 위젯

홈 화면 위젯을 사용할 경우 앱은 시스템이 허용한 App Group 컨테이너에 제한된 항목 요약 데이터를 기록합니다. 이 데이터는 기기에만 남고 외부 서버로 업로드되지 않습니다.

## 7. 내보내기, 가져오기, 공유

내보내기를 사용하면 앱이 기기에 JSON 파일을 생성합니다. 어디에 저장하고 어떻게 공유할지는 사용자가 직접 결정합니다. 가져오기는 사용자가 명시적으로 선택한 JSON 파일만 읽습니다.

## 8. 외부 링크

AwaitGoods에는 상품 링크를 저장할 수 있습니다. 링크를 열면 타사 웹사이트나 앱이 실행될 수 있으며, 해당 서비스는 각자의 개인정보 처리방침을 따릅니다.

## 9. 문의 시 제공되는 정보

이메일이나 WeChat으로 지원을 요청하면, 사용자가 직접 제공한 연락처, 문제 설명, 스크린샷, 기기 정보 등을 받게 됩니다. 이 정보는 답변, 피드백 처리, 제품 개선에만 사용됩니다.

## 10. 데이터 저장과 삭제

AwaitGoods 데이터는 기본적으로 사용자의 기기에 로컬 저장됩니다.

다음 방법으로 데이터를 삭제할 수 있습니다:

- 앱 설정에서 모든 데이터 지우기
- 개별 항목 삭제
- 기기에서 앱 삭제
- 내보낸 백업 파일을 직접 삭제

## 11. 아동의 개인정보

AwaitGoods는 아동을 대상으로 하지 않으며 아동의 개인정보를 고의로 수집하지 않습니다.

## 12. 정책 업데이트

향후 버전에서 계정, 클라우드 동기화, 분석, 서드파티 SDK가 추가되면 이 정책을 업데이트하고 변경 사항을 합리적인 방식으로 안내하겠습니다.
