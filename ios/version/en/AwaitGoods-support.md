## 4. FAQ

### Why does the widget not refresh immediately?

Widget refresh timing is controlled by the system and may not be instant. Open the app to confirm the item is saved, return to the home screen, and wait for the system to refresh, or add the widget again.

### Is my data uploaded to a server?

No. The current version stores data locally by default, does not require an account, and does not provide cloud sync.

### Can I back up my data?

Yes. You can export a JSON file from Settings and import or merge data from a JSON file later.

### How do I delete all data?

Use “Clear All Data” in Settings or delete the app. If you exported backup files, delete those files separately.
