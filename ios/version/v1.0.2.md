# 候物 iOS v1.0.2 更新说明

发布日期：2026-04-29

主要更新

- 改进：将「快捷新增」从底部常驻输入条改为可下滑关闭的底部弹层，避免遮挡列表内容并提升可关闭性与可访问性。
- 功能：在「设置 → 数据」中新增「导入备份文件」，支持 v1 JSON 备份的导入，按 id 覆盖或合并已存在的条目，导入后会提示新增/更新统计。
- 本地化：新增繁体中文（zh-Hant）翻译与语言选择，语言芯片支持横向滚动以避免溢出。
- 修复：修正 Info.plist 中 `CFBundleShortVersionString` 被硬编码为 `1.0` 的问题，改为使用构建配置（`MARKETING_VERSION` / `CURRENT_PROJECT_VERSION`），避免上传到 App Store 时被视为旧 train。
- 其它：若干稳定性与兼容性改进，改善构建与上传体验。

使用建议

- 重新 Archive 并上传新 build；如 App Store 要求更高的 build number，请在 `ios/project.yml` 中把 `CURRENT_PROJECT_VERSION` 提升后再 archive。

技术备注

- 主要改动文件：`ios/AwaitGoods/Views/WishListView.swift`（快捷新增改为 sheet）、`ios/AwaitGoods/Views/SettingsView.swift`（新增导入逻辑）、`ios/AwaitGoods/Models/AppLanguage.swift`（新增繁体映射）、`ios/project.yml`（版本配置为可控来源）。
