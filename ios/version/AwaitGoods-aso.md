## 4. App 描述

候物是一款克制、安静、以等待为核心的购物清单 App。

当你遇到一件很想买的东西，不必马上下单。先把它记进候物，给它一点时间，再决定它是否真的值得。

### 适合谁

- 经常冲动下单，想给自己一点缓冲时间的人
- 想把购物欲望整理成更清楚清单的人
- 喜欢极简、轻量、无打扰工具的人

### 你可以做什么

- 记录想买的物品名称、价格、链接、分类和备注
- 为每件物品设置等待期或目标日期
- 在首页按状态筛选、搜索和整理清单
- 用小组件快速回看仍在等待中的物品
- 通过本地提醒，在等待结束时重新判断
- 导出 JSON 备份，保留自己的购物记录

### 产品特点

- 极简奶油风界面，轻量、不打扰
- 本地优先，无需注册账号
- 支持浅色、深色模式
- 支持 Widget 小组件快速查看
- 支持批量整理和状态流转

### 候物的核心不是买，而是等

有些东西，等一等会更确定；
有些东西，等一等就不想要了。

候物希望帮你把“想买”变成“想清楚再买”。

## 6. 审核备注

可直接提交以下说明给 Apple 审核：

候物 AwaitGoods 是一款本地运行的购物等待清单工具，无账号系统、无付费功能、无第三方登录。

审核时可直接进入首页并测试以下能力：

- 点击右上角加号，新建候物条目
- 在编辑页填写名称后点击保存，条目会出现在首页
- 可为条目设置等待期、目标日期、标记色和备注
- 首页支持搜索、状态筛选、批量整理和状态切换
- 设置页可切换外观模式、通知、导出 JSON 和清空数据
- 小组件用于展示等待中的物品，不包含支付或账号能力

说明：

- App 数据默认仅保存在设备本地
- 通知仅用于本地提醒等待期结束，不用于营销推送
- 商品链接由用户自行填写，点击后会通过系统浏览器打开对应网页
