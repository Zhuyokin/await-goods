## 4. 常见问题

### 4.1 为什么小组件没有立即刷新？

小组件刷新受系统调度影响，通常不会完全实时。你可以尝试：

- 打开 App，确认条目已保存
- 返回桌面等待系统刷新
- 长按小组件重新添加或编辑

### 4.2 数据是否会上传到服务器？

当前版本默认将数据保存在设备本地，不要求注册账号，也不提供云端同步。

### 4.3 我可以备份数据吗？

可以。你可以在设置页导出 JSON 文件，也可以从 JSON 文件导入或合并数据。

### 4.4 我想彻底删除数据怎么办？

你可以在设置页执行“清空全部数据”，也可以直接删除 App。若你曾导出过文件，请自行删除导出的副本。
