## 4. FAQ

### Warum aktualisiert sich das Widget nicht sofort?

Die Aktualisierung von Widgets wird vom System gesteuert und ist nicht immer sofort sichtbar. Öffnen Sie die App, prüfen Sie, ob der Eintrag gespeichert wurde, kehren Sie dann zum Home-Bildschirm zurück und warten Sie auf die Systemaktualisierung oder fügen Sie das Widget erneut hinzu.

### Werden meine Daten auf einen Server hochgeladen?

Nein. Die aktuelle Version speichert Daten standardmäßig lokal, verlangt kein Konto und bietet keine Cloud-Synchronisierung.

### Kann ich meine Daten sichern?

Ja. Sie können in den Einstellungen eine JSON-Datei exportieren und Daten später aus einer JSON-Datei importieren oder zusammenführen.

### Wie lösche ich alle Daten?

Verwenden Sie „Alle Daten löschen“ in den Einstellungen oder löschen Sie die App. Exportierte Backup-Dateien müssen Sie separat entfernen.
