# 候物 iOS v1.0.4 更新说明

发布日期：2026-05-02

主要更新

- 功能：新增首次安装引导页，首次打开 App 时用文字、线条和票据式 UI 介绍候物的记录、存钱进度、整理筛选、桌面小组件和 JSON 备份等基础使用方式。
- 功能：新增艺术感 Launch Screen，打开 App 时使用静态主题图形搭配「候物 AwaitGoods」应用名。
- 交互：引导页语言按钮支持自动轮播和高亮提示，用户可轻触切换显示语言。
- 交互：首页顶部移除「+」新增入口，改为列表右下角圆形悬浮按钮，保留顶部空间给标题、搜索和筛选整理入口。
- 交互：首页搜索框改为右上角搜索图标唤起，默认状态更简洁。
- 交互：优化愿望列表 item 布局，长标题最多展示两行，优先级和价格固定在右侧信息列，避免被挤压变形。
- 交互：设置页新增多套主题配色，可在春日纸笺、海盐蓝调、暮莓花园、墨绿静夜和杏茶暖光之间切换。
- 交互：设置页点击微信联系入口后，除行内状态外新增底部 toast，复制结果更明确。
- 本地化：默认金额符号调整为美元 `$`，调试截图数据改为约 50 个英文热门物品，覆盖奢侈品、Apple 热门款和女性向常见心愿。
- 修复：补充主界面全局小组件同步监听，新增、删除、状态变化和设置页清空/导入后会更可靠刷新小组件数据。
- 本地化：小组件 App 名称与小组件展示文案支持简体中文、繁體中文和 English，并会跟随 App 内语言设置同步。
- 文档：新增 `zh-Hans`、`zh-Hant`、`en`、`ja`、`fr`、`de`、`it`、`ko` 八套语言目录，每套包含截图文案、ASO 文案、隐私政策和技术支持页。
- 版本：设置页「关于 App」版本展示更新为 v1.0.4，并同步 `ios/project.yml` 中 `MARKETING_VERSION`。

使用建议

- 首次安装会展示引导页；点按「开始使用」后会进入主界面，并通过 `AppStorage` 记住完成状态。
- 如需在开发环境反复查看引导页，可清除模拟器 App 数据或删除 `hasSeenOnboarding` 存储值。
- 若后续继续使用 XcodeGen，请在修改 `ios/project.yml` 后执行 `xcodegen generate --spec ios/project.yml` 同步工程。
