## 4. 常見問題

### 為什麼小工具沒有立即刷新？

小工具刷新受系統調度影響，通常不會完全即時。你可以開啟 App 確認條目已保存，然後回到桌面等待系統刷新，或重新添加小工具。

### 資料是否會上傳到伺服器？

目前版本預設將資料保存在裝置本地，不要求註冊帳號，也不提供雲端同步。

### 我可以備份資料嗎？

可以。你可以在設定頁匯出 JSON 檔案，也可以從 JSON 檔案匯入或合併資料。

### 我想徹底刪除資料怎麼辦？

你可以在設定頁執行「清空全部資料」，也可以直接刪除 App。若你曾匯出過備份檔案，請自行刪除匯出的副本。
