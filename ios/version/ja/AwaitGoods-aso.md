## App 説明

AwaitGoods は、願いごとリスト、ほしいものリスト、買い物リスト、貯金進捗のための軽いツールです。

欲しいものを見つけたとき、すぐに買う必要はありません。まず AwaitGoods に入れて、価格、貯金額、カテゴリ、リンク、メモを追加します。進捗、状態、優先度を見ながら、本当に近づきたいものかどうかをゆっくり判断できます。

こんな用途に向いています：

- 欲しいガジェット、服、暮らしのもの、ギフト、本や映像作品を記録
- 個人の wishlist や買い物リストを管理
- 貯金額と残り金額を見ながら購入タイミングを考える
- 衝動買いや重複購入を減らす
- アカウント不要、本地保存、静かなリスト体験を保つ

主な機能：

- 価格、貯金額、カテゴリ、リンク付きで素早く追加
- ホームで状態、優先度、貯金進捗を表示
- 詳細画面で貯金額を追加、または一括で満額に
- 欲しい、所有済み、手放すの状態管理
- 右上アイコンから開く検索、絞り込み、ドラッグ並び替え、一括整理
- wishlist の概要と貯金進捗を表示するホーム画面ウィジェット
- JSON の書き出しと読み込み
- 簡体字中国語、繁体字中国語、英語、日本語、フランス語、ドイツ語、イタリア語、韓国語に対応
- ライト、ダーク、システム連動の外観モード
- 複数の上品なテーマ配色

AwaitGoods は EC サービスでも複雑な家計簿でもありません。静かな wishlist として、まず欲しい気持ちを置いて、時間をかけて判断し、最後に本当に持ちたいものだけを生活に迎えるための App です。

## 審査メモ

AwaitGoods は、本地保存の wishlist / 買い物リストユーティリティです。アカウント、課金機能、第三者ログインはありません。

審査では、フローティング追加ボタンから候物を作成し、右上アイコンから検索を開き、絞り込みや並び替え、詳細表示、貯金額の追加、状態切り替え、JSON 書き出し、ウィジェット表示、言語切り替え、テーマ切り替えを確認できます。

App データは既定で端末内に保存されます。商品リンクはユーザーが入力し、開くとシステムブラウザで表示されます。
