## 2. サポート範囲

- App が開かない、またはクラッシュする

- 貯金進捗の表示がおかしい
- ウィジェットが表示されない、または更新されない
- JSON の読み込みまたは書き出しに失敗する
- 言語、ダークモード、テーマ配色、画面表示、操作に関する問題

- 製品名：AwaitGoods
- プラットフォーム：iPhone と iPad
- 必要環境：iOS 17.0 以降
- 現在のバージョン：1.0.4
- 対応言語：簡体字中国語、繁体字中国語、英語、日本語、フランス語、ドイツ語、イタリア語、韓国語

## 4. よくある質問

### ウィジェットがすぐに更新されないのはなぜですか？

ウィジェットの更新タイミングはシステムにより調整されるため、完全にリアルタイムではない場合があります。App を開いて項目が保存されていることを確認し、ホーム画面に戻ってしばらく待つか、ウィジェットを追加し直してください。

### データはサーバーにアップロードされますか？

いいえ。現在のバージョンでは、データは既定で端末内に保存され、アカウント登録やクラウド同期はありません。

### データをバックアップできますか？

はい。設定画面から JSON ファイルを書き出し、後で JSON ファイルから読み込みまたは統合できます。

### すべてのデータを削除するには？

設定画面の「すべてのデータを削除」を使うか、App を削除してください。書き出したバックアップファイルは別途削除してください。

## 5. フィードバック時にあると助かる情報

より早く状況を把握するため、可能であれば次の内容を添えてください。

- 問題が起きた画面と、その前の操作手順
- 毎回再現するかどうか
- 端末モデルと iOS バージョン
- App バージョン
- スクリーンショットや画面収録

## 6. 返信について

できるだけ妥当な時間内に確認と返信を行います。祝日や営業時間外は返信が遅れる場合があります。
