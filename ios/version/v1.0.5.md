# 候物 iOS v1.0.5 更新说明

发布日期：2026-05-05

主要更新

- 功能：新增回收站。首页顶部右侧增加回收站入口，删除候物会先移入回收站，用户可在回收站中恢复、彻底删除或清空回收站。
- 交互：删除候物增加缓冲动效，确认后先展示移入回收站反馈，再从当前列表中移除，减少突兀感。
- 交互：心愿状态变更新增不同反馈特效，切换为「想买」「已拥有」「放下」时分别展示对应图标、色彩和短暂动画。
- 交互：详情页状态按钮与存钱补满后的自动状态变更同步使用同一套状态动效。
- 数据：主列表、状态统计、小组件同步和导出备份默认忽略回收站中的候物；清空全部数据仍会删除所有本地记录。
- 版本：设置页「关于 App」版本展示更新为 v1.0.5，并同步 `ios/project.yml` 中 `MARKETING_VERSION`。

使用建议

- 误删候物后，可点首页顶部右侧回收站图标进入回收站，再点「恢复」回到主列表。
- 回收站内「彻底删除」和「清空」不可恢复，适合在确认不再需要这些记录后使用。
- 若后续继续使用 XcodeGen，请在修改 `ios/project.yml` 后执行 `xcodegen generate --spec ios/project.yml` 同步工程。

技术备注

- 主要改动文件：`ios/AwaitGoods/Models/WishItem.swift`（回收站状态）、`ios/AwaitGoods/Views/WishListView.swift`（回收站入口、软删除、恢复与状态/删除动效）、`ios/AwaitGoods/Views/WishDetailView.swift`（详情页状态动效）、`ios/AwaitGoods/Views/Components/StatusBadge.swift`（共享变更动效组件）、`ios/AwaitGoods/Views/SettingsView.swift`（版本展示、备份导出范围）、`ios/AwaitGoods/Services/WidgetSyncService.swift`（过滤回收站数据）、`ios/AwaitGoods/Models/AppLanguage.swift`（回收站文案）、`ios/project.yml`（版本号）。
