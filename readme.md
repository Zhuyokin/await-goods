# AwaitGoods

AwaitGoods（候物）是一款面向 iOS 17.0 及以上系统的本地优先心愿清单与购物决策 App。产品强调克制、安静、轻量的记录体验，帮助用户先保存“想买”，再根据价格、已存金额、状态和优先级慢慢判断是否值得入手。

## 当前产品基线

- 开发语言：Swift
- 最低系统版本：iOS 17.0
- 支持小组件、主题切换、多语言、深色模式、新手引导页与 Launch 启动页
- 支持 JSON 离线导入 / 导出、技术支持页、隐私政策页与联系客服能力
- 关于页面需包含 App 介绍、版本信息、微信 `zhuyokin` 复制与 Toast 提示、主题设置、多语言切换、离线数据导入 / 导出入口

## 文档结构

- `docs/ios-app-skills.md`：工程级产品能力与维护规范
- `docs/<locale>/`：每个语种独立维护技术支持、隐私政策、ASO 文档和 App Store 截图文案
- `versions/`：按 `v1.0.1.md`、`v1.0.2.md` 这类格式维护版本更新记录
- `ios/version/`：历史需求与旧版上架文案沉淀目录，后续不作为根级协作文档主入口

当前已维护语种：`zh-Hans`、`zh-Hant`、`en`、`ja`、`ko`、`de`、`it`、`fr`

## 版本与文档同步规则

每次版本更新时，必须同步完成以下动作：

1. 更新或新增 `versions/vx.y.z.md`
2. 校对受影响语种的 `docs/<locale>/AwaitGoods-aso.md`
3. 若功能影响支持、隐私、截图卖点或语言范围，同步更新对应语种文档
4. 更新本 `readme.md`，确保功能基线、目录入口和当前文档维护方式一致
