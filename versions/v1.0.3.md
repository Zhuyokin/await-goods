# 候物 iOS v1.0.3 更新说明

发布日期：2026-05-01

主要更新

- 功能：首页候物列表新增优先级小标签，用低/中/高文本和不同颜色快速区分当前心愿的轻重缓急。
- 改进：首页 item 半屏操作弹窗中，「复制链接」完成后新增底部成功提示，并保留轻触反馈，让操作结果更明确。
- 功能：分类建议改为动态生成，除默认分类外，会自动合并已有候物中使用过的分类；在编辑页或快捷新增里输入新的非预设分类后，该分类会立即出现在可选项中。
- 改进：编辑页和快捷新增统一使用分类去重与空白清理逻辑，减少重复分类和前后空格导致的选中状态不一致。
- 版本：设置页「关于 App」版本展示更新为 v1.0.3，用于与本次体验优化版本保持一致。
- 工程：补充 XcodeGen 中 Info.plist 版本字段、App Store 分类、签名团队和产品名配置，避免重新生成工程时覆盖关键元数据。

使用建议

- 如需提交 App Store，请确认 `ios/project.yml` 中的 `MARKETING_VERSION` 与本次上线版本一致；若 App Store Connect 要求新的 build number，请同步提升 `CURRENT_PROJECT_VERSION` 后重新 Archive。
- 动态分类来自本地已有数据，用户删除或修改条目后，分类建议会随当前数据自然变化。

技术备注

- 主要改动文件：`ios/AwaitGoods/Views/WishRowView.swift`（首页优先级标签）、`ios/AwaitGoods/Views/WishListView.swift`（复制链接 toast 与快捷新增动态分类）、`ios/AwaitGoods/Views/WishEditorView.swift`（编辑页动态分类）、`ios/AwaitGoods/Models/WishItem.swift`（分类建议生成逻辑）、`ios/AwaitGoods/Views/SettingsView.swift`（版本展示）、`ios/project.yml`（XcodeGen 版本与工程元数据配置）。
- 已执行 `xcodegen generate --spec ios/project.yml` 同步工程，并通过 `xcodebuild -project ios/AwaitGoods.xcodeproj -scheme AwaitGoods -destination 'generic/platform=iOS Simulator' build` 验证构建成功。
